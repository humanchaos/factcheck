# Privacy Policy

**factcheck** is built with a "Privacy by Design" philosophy.

1. **No Tracking:** We do not collect, store, or sell your browsing history, personal data, or metadata.
2. **Local Storage:** Your Gemini API key is stored locally on your device. It is never sent to our servers.
3. **Data Transmission:** Only the text content of the YouTube video you are currently fact-checking is sent to the Gemini API for analysis.
4. **No Cookies:** This extension does not use tracking cookies.
5. **Transparency:** As an open-source project, our code is fully auditable to verify these claims.
